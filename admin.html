Come ricevere webhook

Ogni richiesta POST include header X-Veylon-Signature (HMAC-SHA256 in hex del body con il tuo secret) e X-Veylon-Event con il nome dell'evento. Verifica sempre la firma prima di elaborare il payload.

Esempio Node.js (Express)

import express from 'express';
import crypto from 'crypto';

const app = express();
const SECRET = process.env.VEYLON_WEBHOOK_SECRET;

app.post('/veylon/webhook',
  express.raw({ type: 'application/json' }),
  (req, res) => {
    const sig = req.header('X-Veylon-Signature');
    const evento = req.header('X-Veylon-Event');
    const expected = crypto
      .createHmac('sha256', SECRET)
      .update(req.body)
      .digest('hex');
    if (!sig || !crypto.timingSafeEqual(
        Buffer.from(sig, 'hex'),
        Buffer.from(expected, 'hex'))) {
      return res.status(401).send('invalid signature');
    }
    const payload = JSON.parse(req.body.toString('utf8'));
    console.log('Evento Veylon:', evento, payload);
    res.status(200).send('ok');
  });

app.listen(3000);

Esempio Python (Flask)

import hmac, hashlib, os
from flask import Flask, request, abort

app = Flask(__name__)
SECRET = os.environ['VEYLON_WEBHOOK_SECRET'].encode()

@app.route('/veylon/webhook', methods=['POST'])
def webhook():
    sig = request.headers.get('X-Veylon-Signature', '')
    evento = request.headers.get('X-Veylon-Event', '')
    body = request.get_data()
    expected = hmac.new(SECRET, body, hashlib.sha256).hexdigest()
    if not hmac.compare_digest(sig, expected):
        abort(401)
    payload = request.get_json()
    print(f'Evento Veylon: {evento}', payload)
    return 'ok', 200

Note importanti

  • L'endpoint deve rispondere con HTTP 2xx entro 10 secondi.
  • Le risposte non-2xx vengono ritentate fino a 5 volte con backoff esponenziale.
  • Il body e' sempre JSON con campi: evento, timestamp, organization_id, data.
  • Il secret e' visibile una sola volta alla creazione: salvalo in sicurezza.
  • Usa sempre timingSafeEqual/compare_digest per evitare timing attack.
