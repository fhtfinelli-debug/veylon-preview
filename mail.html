Verifica configurazione...

Configura Veylon come server SMTP outbound nel tuo client di posta. Tutte le email che mandi passano attraverso Veylon (che firma + ancora + inoltra al destinatario) e tu ricevi tutte e 3 le ricevute: Invio, Consegna alla casella SMTP, Contenuto Sigillato. Il mittente nominale resta il tuo From abituale.

1
Genera credenziali SMTP personali (username + password applicativa):
Le credenziali sono valide solo per il tuo account, mai condividerle.
Host smtp.veylon.app
Porta 587 (STARTTLS)
Sicurezza STARTTLS
Username —
Password —
From tua@email.com (resta la tua)
2
Configura il tuo client:
  • Gmail (web): Impostazioni → Account → Aggiungi un altro indirizzo → usa il server SMTP Veylon
  • Outlook desktop: File → Impostazioni account → Modifica server uscita SMTP
  • Apple Mail: Preferenze → Account → Server posta in uscita → modifica
  • Thunderbird: Impostazioni account → Server posta in uscita (SMTP) → modifica
3
Manda le tue email come sempre, dal tuo client preferito. Veylon le firma e le inoltra al destinatario, e ti recapita 3 ricevute certificate.
Stato: Backend Relay parzialmente operativo. L'invio HTTP (via API JSON) funziona. Il gateway SMTP vero (host smtp.veylon.app) richiede ancora setup container Postfix o passaggio a Mailgun Submission API. Vedi PROGRESS.md sezione N-Mail.NM-B.1 per le opzioni.
