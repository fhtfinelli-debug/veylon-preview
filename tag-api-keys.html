Non hai ancora chiavi API

Le chiavi API ti permettono di chiamare Veylon dal tuo backend: crei un tag, emetti un certificato, invii una mail certificata, tutto via REST.

Genera la prima chiave, scegli gli scope (cosa puo' fare) e copiala in un posto sicuro: la vedrai una sola volta.

Esempi di codice

// Test connessione
const r = await fetch(
  "https://api.veylon.app/v1/me",
  {
    headers: {
      Authorization: `Bearer ${VEYLON_KEY}`
    }
  }
);
console.log(await r.json());
import os, requests

key = os.environ["VEYLON_KEY"]
r = requests.get(
  "https://api.veylon.app/v1/me",
  headers={"Authorization": f"Bearer {key}"},
)
print(r.json())
curl -H "Authorization: Bearer vlk_YOUR_KEY_HERE" \
     https://api.veylon.app/v1/me

Revocare la chiave?

Le integrazioni che usano questa chiave smetteranno di funzionare immediatamente. La revoca e' definitiva: la chiave non potra' essere riattivata.
